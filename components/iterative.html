Retrofit: Iterative Design

Timeline: April 2025
Skills: Figma, Balsamiq, HTML, CSS
Team: Kristen Cai, Malin Leven, Austin Williams, Catherine Yu
Individual Contribution: Home page wireframe, demo videos, home page hi-fi

Overview

Iterative design is a process that creates better products based on continuous feedback from users and stakeholders. My team designed an interactive mobile prototype for Retrofit, a startup.

As the vintage and secondhand market grows, its online marketplace has become more overwhelming. Retrofit's solution is an AI-curated vintage marketplace, which uses social media trends, sales data, and a user's style to personalize a user's shopping experience.


Sketching

We began with baseline sketches for the application.


Wireframing

We created mobile application mock-ups from our sketches, reflecting a design that is both convenient and user-friendly.

Here is a link to a video detailing the thought process behind the wireframe.


Hi-fi Prototype

After receiving feedback from a stakeholder, we used Figma to design a hi-fi prototype, making some changes:

  • Added checkout, search results, and brand specific screens
  • Created add to cart and percent match informational popups
  • Added private/public distinction to profile page
  • Changed grid on home page to sectioned recommendations

Here is the link to our full interactive Figma prototype, and a video here demonstrating the interface.


User Testing

Users tested our design by searching for an item and completing the checkout process.

  • Users found the interface very intuitive due to its similarity to other shopping interfaces.
  • Users were able to easily find the correct buttons/interactive components to press to work through the task.
  • Some users were confused when searching so a next step would be to add a keyboard to the prototype.
  • Some users didn't like the persistent popup when adding an item to cart, so a next step would be replacing it with a banner at the top of the app, allowing the user to continue their shopping momentum.

Conclusion

Through our iterative design process, we developed an intuitive, user-centric mobile app for Retrofit. By incorporating user feedback as much as possible, we created an interface that effectively integrates AI-curation into a seamless browsing and purchasing experience.
